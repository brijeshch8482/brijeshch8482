## 🚂 What I build

**FogPASS — Indian Railways Pilot Assistance Device** *(Matsushi Power Technologies)*
Sole developer of a device that helps loco pilots navigate safely in dense fog.
**v1 is in production — running in the field for 8+ months.** Now building FogPASS 3.0.

- 📡 Dual-GPS navigation — native **C++ (NDK) UART driver** + automatic failover
- 🧭 **Dead reckoning** (EKF: GNSS speed + IMU + route snapping) — works inside tunnels, zero satellites
- 🔀 Two-phase geometric **track-change detection** — detects route diversion at junctions in real time
- 🗣️ **Fully offline neural TTS** (Hindi + English) on a no-Google tablet — ONNX conversion & latency tuning
- 🔒 Platform-signed **kiosk (Device Owner)** system app + web admin panel + REST backend (Node.js · PostgreSQL · AWS)
